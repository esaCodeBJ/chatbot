
### Projet : Mohamed Chatbot | Esacode

**Description :**  
Ce projet est une application web de chatbot interactive, conçue pour fournir une interface conviviale pour la communication avec un chatbot. Le projet comprend une page web (`index.html`), un script JavaScript (`script.js`) et une feuille de style (`style.css`).

**Fichiers :**
1. `index.html` - La structure principale de la page web du chatbot.
2. `script.js` - Le script JavaScript qui gère les interactions avec le chatbot, y compris l'envoi et la réception de messages.
3. `style.css` - La feuille de style CSS qui définit le style et la mise en page de la page web.
4. `bot.png` - Une image utilisée dans l'interface du chatbot.

**Caractéristiques Principales :**
- Interface chatbot interactive.
- Fonctionnalités de réponse automatique basées sur un dictionnaire de phrases prédéfinies.
- Conception responsive adaptée aux différents appareils.

**Comment Utiliser :**
1. Ouvrez `index.html` dans un navigateur pour lancer l'application.
2. Utilisez l'interface pour interagir avec le chatbot.

**Dépendances Externes :**
- Google Fonts pour les icônes et les polices.

**Note de Sécurité :**
Le fichier `script.js` contient une clé API. Assurez-vous de protéger cette clé dans un environnement de production.
